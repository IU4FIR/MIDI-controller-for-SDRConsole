# Arturia BEATSTEP

## https://www.arturia.com/products/hybrid-synths/beatstep/overview
![BeatStep_111813 TOP](https://user-images.githubusercontent.com/96939950/148024423-9f36837d-5712-46e9-8975-26f0ba584473.png)
